# QuantFolio Analyzer

A quantitative financial portfolio analysis project demonstrating data analysis skills with Python, SQL, and Power BI.

## 🚀 Quick Start

### For Viewers (GitHub Browsing)
Simply browse the notebooks in order - they work with included sample data:
1. `notebooks/01_data_collection.ipynb` - Data downloading logic
2. `notebooks/02_eda_cleaning.ipynb` - Exploratory data analysis

### For Developers (Local Execution)
```bash
# 1. Clone and setup
git clone https://github.com/yourusername/QuantFolioAnalysis.git
cd QuantFolioAnalysis

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run notebooks in order:
#    - Start with 01_data_collection.ipynb to get fresh data
#    - Then run 02_eda_cleaning.ipynb for analysis
